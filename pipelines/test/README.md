# Test Pipeline Configuration

This directory contains the test pipeline definitions and configurations.

## Test Pipeline Stages

1. **Setup**: Prepare the test environment
2. **Unit Tests**: Run unit tests with coverage
3. **Integration Tests**: Run integration tests
4. **Reports**: Generate test reports and coverage data

## Test Types

- Unit tests: Fast, isolated tests
- Integration tests: Tests with external dependencies

## Failure Handling

On test failure, logs are captured and the AI fixer analyzes the failures to suggest fixes.
